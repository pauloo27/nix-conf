# Niri Keybinds Cheatsheet

## Focus (navigate)
| Bind | Action |
|------|--------|
| `Mod+H/J/K/L` | Focus column left / window down / window up / column right |
| `Mod+Arrows` | Same as above |
| `Mod+Home/End` | Focus first / last column |
| `Mod+U/I` | Focus workspace down / up |
| `Mod+Page_Down/Up` | Focus workspace down / up |
| `Mod+1-9` | Focus workspace by index |
| `Mod+Shift+H/J/K/L` | Focus monitor left / down / up / right |
| `Mod+Shift+Arrows` | Same as above |
| `Mod+O` | Toggle overview |

## Move windows
| Bind | Action |
|------|--------|
| `Mod+Ctrl+H/J/K/L` | Move column left / window down / window up / column right |
| `Mod+Ctrl+Arrows` | Same as above |
| `Mod+Ctrl+Home/End` | Move column to first / last |
| `Mod+Ctrl+U/I` | Move column to workspace down / up |
| `Mod+Ctrl+Page_Down/Up` | Same as above |
| `Mod+Ctrl+1-9` | Move column to workspace by index |
| `Mod+Shift+Comma/Period` | Move column to monitor left / right |
| `Mod+Shift+U/I` | Move workspace down / up |
| `Mod+Shift+Page_Down/Up` | Same as above |

## Window layout
| Bind | Action |
|------|--------|
| `Mod+R` | Cycle preset column widths |
| `Mod+Shift+R` | Cycle preset window heights |
| `Mod+Ctrl+R` | Reset window height |
| `Mod+F` | Maximize column |
| `Mod+Shift+F` | Fullscreen window |
| `Mod+Ctrl+F` | Expand column to available width |
| `Mod+C` | Center column |
| `Mod+Ctrl+C` | Center all visible columns |
| `Mod+Minus/Equal` | Shrink / grow column width by 10% |
| `Mod+Shift+Minus/Equal` | Shrink / grow window height by 10% |
| `Mod+BracketLeft/Right` | Consume or expel window left / right |
| `Mod+Comma` | Consume window into column |
| `Mod+Period` | Expel window from column |
| `Mod+V` | Toggle floating |
| `Mod+Shift+V` | Switch focus between floating and tiling |
| `Mod+T` | Toggle tabbed column |
